# LED: LANGUAGE OF EFFECTIVE DEFINITIONS
LED system translates [literate language LED][LEDspec] into:
- [programming language SequenceL][SL]
- documenting language LaTeX

## EXAMPLE
- [LED input][tttLED]
- [SequenceL output][tttSL]
